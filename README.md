# Life Solutions - Site Institucional

Site estático em HTML, CSS e JavaScript.

## Estrutura

- `index.html` — Home
- `sobre-nos.html` — Sobre nós
- `produtos.html` — Página geral de produtos
- `produtos/stents.html`
- `produtos/baloes.html`
- `produtos/balao-farmacologico.html`
- `produtos/tavi.html`
- `produtos/oclusores.html`
- `aplicacoes.html`
- `contato.html`
- `assets/css/styles.css`
- `assets/js/script.js`
- `assets/fotos/` — pasta para colocar fotos, logo e imagens reais

## Como editar imagens

Coloque suas fotos em `assets/fotos/` e troque os caminhos nos HTML.

Sugestões:
- `logo.png`
- `hero.jpg`
- `stents.jpg`
- `baloes.jpg`
- `balao-farmacologico.jpg`
- `tavi.jpg`
- `oclusores.jpg`

## Observação médica

As especificações técnicas estão como placeholder. Não publique dados médicos reais sem validar com a documentação oficial dos fabricantes.
